Installation

Follow the instructions below to install the Beta

  1. Download Java. At the moment you need to download and install Java manually to play the game
  2. Download Desktop version
  3. Download Android version. (Coming Soon!)

It's easiest to create levels on the computer and the game is made to be played on the phone. In the future an installation package will be released for desktops, and you won't need to install Java manually.

Start Here

You have installed Voider, what do you do now? My advice to you is to

  1. Play some levels. Start with Level 1
  2. Create your own level by watching the tutorial below
  3. Publish your level when it's done :D

The best levels (according to the community) will be available with the release of the game. If you are the creator of such a level you will get the full game for free once it is released. :)
